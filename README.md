# Food-delivery-app 🍔

Hi 👋, welcome to FoodExpress – a modern food delivery web platform!

Hi 👋, I'm Rohit Gupta
🎓 B.Tech Student | 💻 Frontend Developer
🌐 Passionate about building clean & responsive web interfaces

This project showcases a seamless online food ordering system, featuring restaurants, menus, and interactive order pages built with HTML, Tailwind CSS, and JavaScript.

🚀 Features
Modern, responsive design with Tailwind CSS

Restaurant Listings: Browse nearby restaurants and cuisines

Menu Pages: View detailed menus with prices, images, and descriptions

Interactive Cart: Add, remove, and customize orders in real-time

Checkout & Order Tracking: Place orders and track delivery status

Mobile-friendly navigation with hamburger menu

Smooth scroll animations & interactive UI elements
